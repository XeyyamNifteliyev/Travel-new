# GitHub Push Planı

## Məqsəd
Bütün faylları `https://github.com/XeyyamNifteliyev/Travel-new.git` reposuna push etmək.

## Addımlar

### Addım 1: Remote əlavə etmək / dəyişdirmək
- Hazırkı remote-ları yoxlamaq (`git remote -v`)
- `origin`-i `https://github.com/XeyyamNifteliyev/Travel-new.git` olaraq təyin etmək (əgər fərqlidirsə update etmək)

### Addım 2: Bütün faylları stage etmək
- `git add -A` ilə bütün dəyişiklikləri əlavə etmək

### Addım 3: Commit etmək
- Mövcud dəyişiklikləri commit etmək

### Addım 4: Push etmək
- `git push -u origin <current-branch>` ilə remote-a push etmək
